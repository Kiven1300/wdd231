NP Membership Benefits

This level is designed for non-profit organizations and has no fee.

  • Basic directory listing
  • Community event invitations
  • Access to chamber newsletters

Bronze Membership Benefits

This level is ideal for small businesses that want basic chamber visibility.

  • Business directory listing
  • Networking event invitations
  • Member newsletter access

Silver Membership Benefits

This level provides additional promotional opportunities and member discounts.

  • Enhanced directory listing
  • Training event discounts
  • Social media promotion opportunities
  • Eligibility for homepage spotlight ads

Gold Membership Benefits

This level is designed for organizations that want premium visibility and sponsorship opportunities.

  • Premium directory listing
  • Priority homepage spotlight placement
  • Event sponsorship opportunities
  • Advertising opportunities
  • Discounted chamber luncheon access
